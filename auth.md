# List the auth info

  * Authentication 認證 
    * 一般指身份验证，又称 “验证”、“鉴权”，是指通过一定的手段，完成对用户身份的确认
  * Authorization 授權
    * 一般是授权、委托的意思，向… 授予职权或权力许可，批准等意思。
