# List Big Data information

## 將 MySQL 資料，儲存到 S3 的方式
  * 用 pandas 將資料由 MySQL 讀到 DataFrame 中
  * 使用 pandas 的 DataFrame.to_parquet 轉資料
  * 將 parquet 儲存到 S3 中
  * 之後可以由 AWS athena 來讀取
