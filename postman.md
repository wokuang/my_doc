## 心得筆記 
  - 可以將常用的 header 設定 preset, 在 header 那欄來設定
  - 能依據環境設定，使用 {{}} 來輸入變數 
  - 能將 request export 成 curl 的 code, 也能匯入 curl 的 code
  - 每個 request 都可以設定 pre-request and tests part 
  - collection 的部分還滿有趣的，需要時間加以練習。其中還可以寫 loop，可以嘗試看看
